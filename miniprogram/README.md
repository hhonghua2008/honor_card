## 能力（P1–P3，无 PDF）

| 阶段 | 功能 | 状态 |
|------|------|------|
| **P1** | 排版对齐 H5：字距、宋体/黑体、弧形签章、照片遮罩 | ✅ |
| **P2** | 批量：粘贴名单 → 逐张生成并保存到相册（最多 100） | ✅ |
| **P3** | 云同步（微信云开发）+ 本地导入/导出备份 | ✅ 需填 `cloudEnv` |
| PDF | 不做，统一保存图片 | — |

## 用户路径

1. **模板** → 选套模板  
2. **编辑** → 改字 / 换照片 → 实时预览  
3. **导出** → 保存相册 / 预览转发  
4. **批量** → 编辑页点「批量」→ 贴名单 → 批量存相册  
5. **我的** → 草稿；可导出备份；开通云开发后可跨设备同步文案

## 开通云同步（P3，可选）

1. 微信开发者工具 → 云开发 → 开通并创建环境  
2. 数据库新建集合 `projects`（权限：仅创建者可读写）  
3. 把环境 ID 填入 `utils/config.js` 的 `cloudEnv`  
4. 重新编译；「我的」页出现「上传/拉取云端」

未开通时：用「导出备份 / 导入备份」（剪贴板 JSON）换设备。
