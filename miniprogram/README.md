## 产品路径（个人主体）

个人小程序**不能**用 web-view。本版在小程序内原生完成：

1. **模板** — 28 套浏览 / 筛选  
2. **编辑** — 改文字、上传照片、Canvas 实时预览  
3. **导出** — 保存相册 / 预览转发  
4. **我的** — 本地草稿

## 开发

1. 微信开发者工具导入本目录  
2. 服务器域名（downloadFile / request）配置：`https://hhonghua2008.github.io`  
3. 详情 → 本地调试可勾选「不校验合法域名」  
4. 编译后点任意模板进入原生编辑器

同步模板数据（改 H5 `templates.js` 后）：

```bash
node scripts/sync-miniprogram-catalog.mjs
```

## 发布

开发者工具 → 上传 → 公众平台提交审核 → 发布。

审核说明建议：提供奖状模板选择与本地图片合成导出，不涉及支付与用户内容上传服务器。
